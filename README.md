# Responsive_project
This project is a simple, interactive, and responsive Tic Tac Toe (O &amp; X) game built using web technologies. The game is played on a 3×3 grid containing 9 boxes arranged in 3 rows and 3 columns. Two players take turns marking the boxes with “O” or “X”. The design is fully responsive, so it works smoothly on mobile, tablet, and desktop screens.
